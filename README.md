# 【Google Cloud】手を動かして学ぶ Gemini CLI

## はじめに

[gemini-cli github](https://github.com/google-gemini/gemini-cli.git)

## 前提

- Node.js v20以上がインストールされていること
- npmが使用可能であること

## Setup

npmでGemini CLIをインストールします。

```bash
npm install -g @google/gemini-cli
```

```bash
export GOOGLE_CLOUD_PROJECT=your-project-id
```

check [google cloud console](https://console.cloud.google.com/welcome)
